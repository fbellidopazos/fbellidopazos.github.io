Project Idea

The Main idea is to have a unique bot that unites the best funcionalities of others
The program will be running in a host computer (Can be a raspberry,or any computer) or Heroku(Preferably)
To make it work it is required to configure manually the discord bot on discord dev platform

Reason behind it:
I was tired of seeing millions of bots with different commands many of which are useless to the main user , and so the bot was made.A bot that unifies all main commands used by many users.
