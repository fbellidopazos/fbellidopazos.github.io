Project Idea:

The Main idea is to have a central hub(Raspberry Pi) ,in which the program will be running, in order to control the lights and make use of the app.This central hub will have a touch screen in case Alexa is down or not functioning.
After Main hub , the other devices(Raspberry Pi Zero W) will be connected to the light switch via a relay,and in order to turn lights on the program (alocated in the Main Hub)will send a remoteGPIO request in order to turn the lights on/off
To access the app just need to installed it and configure it manually,after that is done you will have access to the appp
